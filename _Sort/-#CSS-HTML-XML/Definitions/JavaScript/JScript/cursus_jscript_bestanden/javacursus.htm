Introductie


Met javascript kun je heel veel verschillende dingen, je kunt berekeningen maken, visuele effecten in je site zetten, meer interactie met de bezoeker etc. Javascript is afgeleid van de programmeertaal C++. Javascript wordt tussen de script tags gezet:

<script language="javascript">
<!--

// -->
</script>

Hiertussen komt de script, de <!-- //--> zorgt ervoor dat oude browsers die geen script ondersteunen de script niet op de pagina scrijven, de // voor de laatste --> zorgt ervoor dat de laatste --> niet als script wordt gezien. Alles op een regel na // wordt niet als script gezien, alles tussen /* en */ wordt ook als commentaar gezien, dus niet als script. De script tags kunnen overal in de pagina voorkomen zolang die maar tussen de HTML-tags staan, het beste is dat je de script tag tussen de head-tags plaatst.

Als we een voorbeeld geven van een script laten we vanaf nu de script-tags niet zien!

Variabelen

Javascript werkt met variabelen, een variabele is bijvoorbeeld een letter die een waarde voorstelt, bijvoorbeeld (bv) i=6, hier heeft de variabele i de waarde 6 gekregen. Maar voordat je gebruikt maakt van variabelen moet je eerst de computer laten zien dat de ee bepaalde letter een variabele is, bv (dit hieronder hoort tussen de script tags:

var i

De var geeft aan dat er een variabele komt, de computer weet nu dat de variabele i bestaat, je kunt er nu mee gaan rekenen. In een Variable mag geen spatie voorkomen, dus var de boom is fout, je kunt dan het beste gebruik maken van hoofdetters dus var DeBoom. Bij dat toekennen (declareren) kun je de variabele ook meteen een waarde geven:

var i = 966

Hier is variabele i gedeclareert en heeft het de waarde 966, nu kun je meer met i doen:

var i = 966
i = 6 + 94
i = 100 - 6

hier wordt op de eerste regel de variabele i gedeclareert en krijgt de waarde 966, daarna krijgt i de waarde 6 + 94, dus 100, daarna krijgt i de waarde 100 - 6, dus 94. Nog een voorbeeld:

var BeginAantal = 60
var EindAantal = 30
var C = 100/(BeginAantal/Eindaantal)

Hierboven wordt eerst de variabelen BeginAantal met een waarde van 60 gedeclareert, daarna de variabele EindAantal met een waarde van 30. Dan wordt de variabele C gedeclareert, De waarde van c is hoeveel procenten EindAantal van BeginAantal is, de haakjes zorgen ervoor dat BeginAantal/EindAantal eerst wordt uitgerekent, de / betekent gedeeld door.

C = 100/(BeginAantal/EindAantal)
C = 100/(60/30)
C = 100/2
C = 50
Hier is EindAantal dus 50 procent van BeginAantal.

Operatoren

Operatoren zijn gewoon tekens die berekeningen kunnen uitvoeren,dus + is een operatoor. Hieronder geef ik een paar operatoren op een rijtje:

  • + = plus
  • - = min
  • / = gedeeld door
  • * = maal
  • ^ = tot de macht, dus 4 ^ 3 = 4 * 4 * 4
  • ()= prioriteits haakjes, deze haakjes geven aan wat eerstmoet worden uitrgerekent
Met deze opereatoren kun je dus heel veel, maar het is veel belangrijker om de taal C++ goed te kennen. Je kunt bijvoorbeeld een berekening met de variabele b doen, bijvoorbeeld:

var b = 10 b = b+b
b+=b
b*=2


Hierboven krijgt b een waarde tien daarna krijgt b de waarde b+b, dus 10+10=20, dan komt er iets wat je nog niet kent, b+=b, hier krijgt b de waarde b+b, dit is veel makelijker dan b=b+b, je kunt dit ook met andere operatoren doen, dus b*=b dit is gelijk aan b=b*2. nog een voorbeeld:

var c = 1
c=c+1
c+=1
c++

hier gebeurt er steeds hetzelfde met c, eerst krijgt c de waarde c+1, dus 1+1, c heeft nu dus de waarde 2, dan krijgt zoals ik hierboven heb uitgelegt c+=1, dus c=c+1, dus c=2+1, c heeft nu e waarde 3, dan gebeurt er iets wat je nog niet kent: c++, hier gebeurt er hetzelfde als met c+=1, c krijgt een waarde hoger, c++, hier staat eigenlijk c+=1, c krijgt nu dus de waarde 4. Je kunt ook c-=1 (dus c=c-1) korter opscrijven, namelijk: c--. Nu weet je ook waarom die programeertaal C++, heet, eerst heette die programeertaal C, maar de nieuwe versie heet C++, ééntje hoger dus.

Zoals je nu de heletijd al gezien heeft gebeurt alles regel voor regel, maar net zoals bij HTML leest ook C++ (javascript) de spaties niet, maar C++ leest de regels wel, als je toch een bepaalde berekening op één regel wilt hebben moet je gebruik maken van het ; teken, dat teken geeft aan dat er iets nieuws komt, een nieuwe berekening ofzo iets:

var a=8

Functies

Een functie is een bepaald gebied (code) die integenstelling tot code die niet in een functie staan niet met het opstarten van een pagina gelezen worden. Een functie kun je tevens aanroepen, dit is nu allemaal een kleinbeetje onduidelijk, maar misschien verheldert een voorbeeld alles:

var x=10

function DezeFunctie() {
x++
}

DezeFunctie()

Hierboven wordt eerst de variabele x ingelezen, x krijgt hier de waardew 10, dan komt er een functie, maar die functie wordt niet ingelezen, dan komt er dit: DezeFunctie(), deze regel roept de functie DezeFunctie aan, in de functie DezeFunctie wordt er bij x 1 opgetelt (x++).

Een functie is op een bepaalde manier opgebouwd:

function x(){}

de tekst function geeft aan dat er een functie volgt, x is de naam van de functie, en de twee () worden later uitgelegd, er staan ook nog twee accolades {}, de accolades geven aan wat er moet gebeuren als de functie wordt aangeroepen, alles tussen de accolades wordt uitgevoerd. Het aanroepen van een functie zodat die fuctie wordt uitgevoerd is ook heel gemakkelijk: geef gewoon de naam van de functie plus de (). een functie kan ook werken met variabelen binnen de functie:

function BerekenInhoud(h,b,l) {
var inhoud = h * b * l
alert("De inhoud van het object ="+inhoud+"cm3")
}

hh = prompt("geef hier de hoogte in cm")
bb = prompt("geef hier de breedte in cm")
ll = prompt("geef hier de lengte in cm")
BerekenInhoud(hh,bb,ll)

Als je de code hierboven zou uitvoeren gebeurt er .
Zoals je ziet zijn de () bij een functie toch wel nuttig, je kunt daar variabele plaatsen: function naam(var1,var2,var4,var4){}. Je ziet dat de variabelen gescheiden zijn door een komma. binnen zo'n functie kun je gebruik maken van de variabelen, bij het aanroepen van een functie moet je die variabelen ook een waarde geven, dus: naam(4,6,2,"hallo"), de variabelen binnen de functie krijgen dan de waarde die ze bij het aanroepen hebben gekregen, dus:

var x = "hallo"
laatzien(x)

function laatzien(tekst) {
alert(tekst)
}

als je dit uitvoert gebeur er dit binnen een functie kun je weer een andere functie aanroepen maar je kunt niet dezelfde functie aanroepen, dan loopt de computer vast, want er wordt dan voor eeuwig door de functie gegaan.

Je kunt binnen een functie ook gebruik maken van return variabele; voorbeeld:

function maal(x){
var z = x * 2
return z;
}

var GG = (2 * maal(4)) / maal(8)


Bij de functie maal(x) is gebruik gemaakt van de return, variabele GG krijgt hier de waarde (2 * maal(4)) / maal(8), het onbekende hieraan is de maal(4) en de maal(8), het zijn eigenlijk niks anders dan twee dingen die een functie aanroepen. bv maal(4) roept de functie maal(x) aan. x krijgt hier de waarde 4, dan krijgt binnen de functie z de waarde x * 2, dus 4 * 2, z is dus gelijk aan 8, door return z krijgt het aanroepen zelf de waarde z, dus de waarde 8. 2 * maal(4) is dus gelijk aan 2 * z omdat z voor de return staat. 2 * maal(4) = 2 * 8 omdat z hier de waarde 8 krijgt door de functie maal(x), dus GG = (2 * maal(4)) / maal(8) = GG = (2 * 8) / 16 = GG = 16 / 16 = GG = 1. Hopenlijk is het een kleinbeetje duidlijk.

Met de while-statement kun je bepaalde stukken javascript heraaldelijk uitvoeren, de opbouw van de while-statement is: while (dit){dat}. Waarbij dat net zo vaak wordt herhaald als dit=true is, voorbeeld:

var i=0

while (i<10){
alert('i = '+i)
i++
}

Hier krijgt eerst i de waarde 0, dan komt de while-statement, er staat eigemlijk: wanneer i kleiner is dan 10 dan { alert('i = '+i) i plus 1. Hierbij wordt de stuk code tussen accolades net zo vaak uitgevoerd zolang i<10 waar is, als er geen i++ tussen de accolades zou staan dan zou je oneindig vaak alert('i = '+i) krijgen. Als je dit hierboven uitvoert zie je steeds een berichte tevoorschijn met een getal, eerst 1,dan 2 dan 3dan 4 en op het eind 5. De break; wordt uitgevoerd als i==5 is.

<input type="" size="" value="" title="" checked event>
Dit hierboven is een input-tag met de meest gebruikte subdefinities:

  • type // type geeft aan wat voor soort het moet zijn, waarden: button, text, radio, checkbox, password.
  • size // geeft de grootte weer
  • value // geeft de waarde van de input weer (de tekst in de button is gewoon de value, zo ook bij de text bij de textbox, en bij password, maar bij radiobox en checkbox zie je niks visueels als je value gebruikt.)
  • title // tooltip (ik hoop dat je deze nog wel kent)
  • checked // checked staat altijd opzich (dus zonder ="") en kan alleen gebruikt worden bij radio en checkbox
  • event // met event bedoel ik gewoon alle events (dus onmousemove onclick enzo)

Met de input Tag kun je nu extra leuke dingen doen, bv:

<input type=button id=X onclick="alert('hallo')" onmousemove="X.value='ja hier'" onmouseout="X.value='Klik hier!'" value="klik hier">

Dit hierboven komt er zo uit te zien:


<div>

Met de div-tag kun je (in Internet Explorer 4 of hoger) iets heel speciaals doen, namelijk gebruik maken van de innerHTML, bv:

<div id=TEL>De div tag</div>

<script language=JavaScript> var i=1
var ja=1
function Tel(){

if(ja==1){
i++
if(i==10) ja=0;
} else {
i--
if(i==1) ja=1;
}
</script>

TEL.innerHTML = "<b>"+i+"</b>"
setTimeout ("kleur()", 500)
}

Als we dit hierboven uitvoeren gebeurt er dit:

de div tag


De functie van hierboven kunnen we iets aanpassen om een veel mooier effect te krijgen:
<div id="Deze">div tag</div>

<script language=JavaScript>
var i=1
var ja=1
var color = new Array ("111100", "222200", "333300", "444400", "565600", "686800", "808000", "909000", "AAAA00", "DDDD00")
var tekst = "Hallo allemaal"

function Kleur(){

if(ja==1){
i++
if(i==10) ja=0;
} else {
i--
if(i==1) ja=1;
}

Deze.innerHTML = "<b><font color="+color[i-1]+">"+tekst+"</font></b>"
setTimeout ("Kleur()", 500)
}
<script>

Als we de functie Kleur() nu aanroepen gebeurt er
div tag


Zo zie je hoe gemakkelijk de div-tag is.

NB, de innerHTML is onderdeel van DHTML (Dynamic HTML), de innerHTML kan bij veel meer tags gebruikt worden maar bij sommige doet hij het niet, daarom kun je beter de div-tag gebruiken (de div-tag wordt oorspronkelijk gebruikt om een paragraaf aan te geven) De innerHTML werkt alleen bij Internet Explorer 4 of hoger!.
