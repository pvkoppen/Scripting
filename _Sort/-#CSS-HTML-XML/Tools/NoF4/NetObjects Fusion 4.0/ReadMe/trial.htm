If you are a trial user

If you are a user of NetObjects Fusion 4.0 Trial, the following applies to you:

 

A note on publishing:

When you publish your site to your hard drive or to a Web server, a footer is added to each page stating that the site was created in a trial version of NetObjects Fusion 4.0. This is done to protect copyrights of NetObjects Fusion trial and pre-release software. This footer is not be generated in the retail version of NetObjects Fusion 4.0.

 

A note on Technical Support:

NetObjects Fusion 4.0 Trial is not actively supported by the NetObjects Technical Support group. If you are a trial user and have questions or are experiencing problems, please visit our Web site to browse our newsgroups, list of frequently asked questions, usage notes or knowledge base. You will find additional information on these sources of help in this ReadMe in the section entitled, "To get help as you begin to build Web sites with NetObjects Fusion 4.0".
