Documentation for NetObjects Fusion 4.0:

NetObjects Fusion 4.0 comes with two guides:

  • The Building Business Web Sites book, included in the NetObjects Fusion 4.0 box, is an extended tutorial that leads you through the basic functionality of NetObjects Fusion. It also helps you create commerce-enabled pages using iCat components and dynamic data-driven pages using Microsoft ASP technology or Allaire Cold Fusion. An electronic copy of the book (in .pdf format) is included on the NetObjects Fusion 4.0 CD and is also available from the NetObjects Web site for download.
     
  •  The NetObjects Fusion 4.0 User Guide covers all functionality of NetObjects Fusion  in detail, providing the most thorough reference material for the product. This guide is provided in .pdf format on the NetObjects Fusion 4.0 CD and can also be downloaded from the NetObjects Web site.

Quick Start Tips:

These tips shorten your learning curve by giving you basic concepts and an overview of the NetObjects Fusion interface, and by pointing you to where you can learn more. The Quick Start Tips document is installed by default when you install NetObjects Fusion 4.0. You can launch the Quick Start Tips HTML document from the NetObjects program group in the Start Menu.
