Problems and Questions


We have tried to make this product as easy to understand and use as possible. However, there are always ways to improve. If you have suggestions, please email them to suggestions@modomedia.com.

If there is a problem with your CD [it is somehow defective] we also want to know that. First consult the website at: http://www.modomedia.com/ . There may be a "fix" posted if there is a problem that is common among users of this product. If you don't find the information you need there, send email to defective@modomedia.com
