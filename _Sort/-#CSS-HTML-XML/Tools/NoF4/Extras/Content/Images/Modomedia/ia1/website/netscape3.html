It is recommended that you use a 4.0 or above Internet Browser to view the IA1 interface. Netscape 4 or Microsoft® Internet Explorer 4 or above will work. You may still enter the image browser with Netscape 3, however, it will not look or operate as good as it would with a more recent Internet Browser.


Microsoft® Internet Explorer 4.01 is provided on the IA1 CD.

Upgrade To 4.0 | Enter Anyway



To Install MSIE 4.0:

  • Exit this browser
  • Make sure that the IA1 CD is in your CD-ROM drive.
  • Click Start (lower left in Windows95), then Run...
  • In the box provided type D:\ie4inst.exe (where "D" is the letter of your CD-ROM drive)
