			CoolMenus is an advanced menu script that I have been working on for years. If you have been browsing this site you have already used the script. The coolMenus have been downloaded over 100 000 times! <BR><BR><STRONG>Please note:</STRONG> The new version called 4.0_beta1 is still in BETA. I am sorry to do this, but I had to get out the updates to the site (new backend) so I had to add this new content. The content in this section is still under development and so is the script. Hopefully I'll have a version that I can call 4.0 period later this week. <BR><BR>See <A href="http://www.dhtmlcentral.com/projects/news.asp?m=30">latest news</A> for info about the next version and support - Updated 5th february.

		<h3 class="clHeading">Main features<br>
		<span class="clLine"></span></h3>
		
    <p class="clWinText">
			- Unlimited submenus<BR>- 99% Browser compatibility (the web can be so tricky that I can't say 100%)<BR>- Very customizable<BR>- Very fast (considering the browser compatibility)
		</p>
  
		<h3 class="clHeading">News and changes in version 4<br>
		<span class="clLine"></span></h3>
		
    <p class="clWinText">
			- It now actually works cross-browser. <BR>- Arrow support added (will be automaticly shown if there are submenus)<BR>- The coolmenus.js file is now a lot lighter, from now on I'll call that the core file and I'll be adding new features to another js file. That means that you more then before can have only the parts of the code you need.<BR>- <STRONG>NOTE: </STRONG>This version does NOT support frames yet. I will be adding that soon, but probably a little different then in version 3 because the frame stuff was the main reason the script was unstable on some systems.<BR>- You no longer set the colors or style in the script. You make CSS classes in a stylesheet that you tell the script to use. You can set classes on levels or on individual menu items. This makes it easier to customize the look of the menu. It does however require some css knowledge.<BR><BR>Everyone that uses coolMenus 3 WITHOUT frames should upgrade as soon as possible. Though it does require some tweaking of the settings of the script to upgrade I assure you it's worth it. See the <A href="/projects/coolmenus/tutorials.asp?m=59">Upgrading to CoolMenus 4</A> tutorial for help.<BR>
		</p>
